## ℹ️ 1. 프로젝트 소개 (About Project)
도시화가 급격히 진행됨에 따라 **흡연구역, 쓰레기 수거함, 전기차 충전소** 등 주민들의 일상에 필수적이면서도 극심한 갈등을 유발하는 **NIMBY(기피) 인프라의 입지 선정 갈등**이 심화되고 있습니다.

본 플랫폼(SDSS)은 공공 데이터의 객관적 분석(정량)과 가상 시뮬레이션 기반 이해관계자 협의(정성)를 결합한 **2단계 하이브리드 의사결정 파이프라인**을 제공하여, 과학적이고 투명한 도시 계획 행정을 어시스트합니다.

*   **기획 피벗 비전**: 스마트시티 기피/갈등 유발 인프라 통합 플랫폼 포지셔닝.
*   **MVP 실증(PoC) 범위**: 7주 완성을 위해 가장 갈등이 빈발하는 **실외 흡연 시설(부스)**을 시범 타겟으로 적용하여 실효성 검증.

## 🏗️ 2. 핵심 아키텍처 및 파이프라인 (Architecture & Pipeline)

### 2.1 2단계 의사결정 워크플로우
*   **1단계: 정량적 입지 최적화 (MCDA-AHP)**: 법적 차단 금연구역 버퍼 연산 및 보행 접근성 감쇄 함수, 안전망 인프라 데이터 중첩 연산 ➔ 최적 후보지 TOP 3 선정.
*   **2단계: 정성적 갈등 시뮬레이션 (Multi-Agent)**: 1단계 후보지 데이터를 입력받아 가상의 페르소나 에이전트(시민대표, 도시계획관, 보건행정관) 간의 3라운드 정책 토론 RAG 시뮬레이션 수행.

### 2.2 클라우드 마이크로서비스 컨테이너 (MSA)
*   **Frontend Pod**: React + Nginx 기반 지도 시각화 및 대시보드.
*   **Backend API Pod**: FastAPI 기반 AHP 연산 및 REST API 서빙.
*   **Celery Worker & Redis**: 분 단위가 소요되는 LLM 연산 작업을 비동기 작업 큐로 백그라운드 처리하여 브라우저 타임아웃(Gateway Timeout) 완벽 차단.
*   **PostGIS DB Pod**: 공간 지오메트리 DB 구축 및 GIST 인덱스를 통한 고속 공간 쿼리 보장.
*   **vLLM Inference Pod**: GPU 전용 노드 풀에서 Solar/Llama-3 sLLM 추론 서비스 담당.

## 📊 3. 활용 데이터셋 및 수집 가이드 (Data Architecture)
플랫폼 연산의 정밀도를 담보하기 위해 **100% 무료 공공데이터포털 및 지자체 Open API 기반 14대 데이터셋**을 연동합니다.

1.  **규제 배제 레이어**: 금연구역 정보, 어린이집 정보, 어린이보호구역 스쿨존 표준 데이터.
2.  **수요 핫스팟 레이어**: 대중교통 역사/정류소 위치, 버스/지하철 탑승 승객 통계, 행정동 생활인구, 상가상권 정보.
3.  **안전 및 위생 인프라**: 스마트 쓰레기통 현황, 담배꽁초 수거함 정보, 행정동 공간 경계 SHP, 연속지적도.
4.  **실증 및 RAG 자료**: 용산구 담배꽁초 상습 무단투기지역(그라운드 트루스), 지자체 금연 조례 전문 텍스트.

➔ *상세한 다운로드 및 가공 가이드는 [[공공데이터_수집_명세서]]를 참고하세요.*

## 📅 4. 개발 로드맵 및 마일스톤 (Milestones)
*   **1주차**: 아키텍처 및 DB 스키마 설계, FastAPI-Docker 기본 셋업
*   **2주차**: 데이터 수집 파이프라인 완료, GeoPandas EPSG:4326 변환 스크립트 구축
*   **3주차**: MCDA-AHP 수리 연산 구현 및 LangChain Multi-Agent 토론 로직 구현
*   **4~5주차**: FastAPI 백엔드 + Kakao Map 프론트엔드 연동, Celery 비동기 큐 안정화
*   **6주차**: 성능 검증 및 최적화 (Model Hit Rate@10 및 F1-Score 평가)
*   **7주차**: 원클릭 PDF 보고서 반출 모듈 완성, Helm Chart 패키징 완료

## 📈 5. 비즈니스 임팩트 및 기대효과 (Business Impact)
*   **예산 매몰 방지**: 주민 민원으로 이전/철거되는 부스(개당 2,500만 원) 낭비를 사전에 예측하여 지방 재정 건전성 극대화.
*   **행정 효율화**: 관련 부서 간 대면 서류 협의로 평균 30일 이상 지연되던 설치 심사를 **1일로 대폭 단축 (시간 96% 절감)**.
*   **보건 비용 절감**: 공공 보건 예산 낭비를 차단하고 시민 간접흡연 피해 노출률을 감소시킴.
