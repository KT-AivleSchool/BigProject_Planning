## 🏁 2. PM 배종현의 개발 통제 및 코드 리뷰 3대 철칙 (Code Review Rules)

1.  **공간 인덱싱(GIST) 누락 시 PR 반려:** 지적도 및 주요 공간 geometry 테이블 생성 쿼리(DDL)에 GIST 인덱스 선언이 제외된 경우 즉시 반려 처리할 것.
2.  **비동기 API(`async def`) 미적용 시 PR 반려:** FastAPI의 API 라우터 함수가 동기식(`def`)으로 선언되어 이벤트 루프를 블로킹하는 코드는 발견 즉시 반려 및 리팩토링 지시할 것.
3.  **AHP C.R. < 0.1 차단 조건 해제 시 PR 반려:** 프런트 및 백엔드 단에서 일관성 비율(C.R.) 검증 조건(0.1 이하)을 우회할 수 있도록 설정된 임시 코드는 형상 관리 머지(Merge)를 불허할 것.
