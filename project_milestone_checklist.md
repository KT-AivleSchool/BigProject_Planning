## ⚖️ 3. AHP 의사결정 파트 체크리스트 (최승헌, 장천명)
*   [ ] **AHP 9점 척도 가중치 산출 로직 구현**:
    *   [ ] 사용자가 슬라이더로 조정한 5대 입지 요인의 역수 쌍대행렬을 기반으로 기하평균 정규화법을 거쳐 최종 가중치 벡터 $w$ 산출 모듈 작성 (`ahp_service.py`).
*   [ ] **AHP 일관성 비율(C.R. < 0.1) 수학적 검증 엔진 탑재**:
    *   [ ] 입력 행렬의 최대 고윳값($\lambda_{max}$) 및 일관성 지수($C.I.$), 무작위 지수($R.I.$)를 계산해 $C.R. = C.I. / R.I.$가 0.1 미만인지 검증하는 통계 알고리즘 구현.
*   [ ] **가중치 DB 모델 락 (AHP Lock)**:
