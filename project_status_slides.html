  • 최종 기술명세서: 13종 테이블 물리 설계 DDL 및 데이터 파이프라인 시퀀스 다이어그램 확정.

  • 학술/특허 타당성: 건축법 가설건축물 신고, 도로법 도로점용, 보도 잔여폭 1.2m 검증식 포함 BM 특허 청구항 설계 완료.

  • 규격 검토서: 3m x 2m 10인 부스 설치 및 소방로 3m 유효폭 확보(도로폭 5m 이상 필지) 강제화에 따른 최종 top3 적격지 갱신.

03. End-to-End 행정 워크플로우

지자체 보안망을 고려한 오프라인 CSV 임포트 기반 4단계 여정

04 / 05
Phase 1 & 2

CSV 감리 및 지능형 필터

공무원이 내부망 수집 CSV를 드래그 업로드하면 1세트 AI 데이터 감리 에이전트가 정제한 후, PostGIS GIST 공간 연산을 통해 위법/시공 불가 필지를 0.5초 이내로 사전 배제합니다.

Phase 3

AHP Lock & AI 조례 토론

AHP 일관성 비율(C.R. < 0.1) 수학적 검증을 통과한 가중치만 저장(Lock)하여 게리맨더링을 차단하고, 2세트 LangGraph RAG 에이전트가 실제 조례를 RAG로 대조하며 주민 합의서를 작성합니다.

Phase 4

행정 PDF 리포트 인쇄

지도 분석, AHP 점수, AI 합의서에 추가로 가설건축물 신고 요건 자가진단표 및 도로점용료 자동 산식을 WeasyPrint로 결합하여 1일 이내로 즉시 다운로드 가능한 결재 공문서를 출력합니다.

04. 차주 마일스톤 및 마이그레이션 로드맵

2주차 스프린트: PostgreSQL / PostGIS DB 구현 및 DDL 마이그레이션

05 / 05
Sprint Task 1

PostGIS DB 서버 구축

Docker Compose를 가동하여 PostgreSQL 및 PostGIS 서버 환경을 로컬에 격리 빌드하고, 연속지적도 및 공공 지리 정보 테이블을 위한 13종 DB 스키마 뼈대를 확보합니다.

Sprint Task 2

ETL 데이터 적재 파이프라인

공무원 CSV 데이터(용산구 PNU 정보, 기존 AED/금연구역, 상가, 야간 생활인구)를 SQLAlchemy 기반으로 PostGIS 기하 객체로 형변환(Fuzzy Geocoding)하여 적재하는 파이썬 코드를 구축합니다.

Sprint Task 3

React GIS 맵 컴포넌트 연동

Next.js(React) 단에서 Mapbox GL JS 라이브러리를 바인딩하여 기본 맵을 연동하고, 백엔드의 공간 데이터를 조인할 기본 프론트엔드 UI 대시보드 템플릿 세팅을 완료합니다.
