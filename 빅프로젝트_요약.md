# 🏙️ 빅프로젝트 요약: OmniSite 플랫폼

본 문서는 **"보건소 소관의 다목적 스마트시티 공간 의사결정시스템 (OmniSite)"**의 핵심 기획, 2차 코칭 피드백 반영 사항, KT 그룹사 비즈니스 시너지, 리스크 방어 예외 처리 및 향후 로드맵을 통합 요약한 옵시디언 노트입니다.

더 상세한 내용은 원본 노트를 참고해 주세요. (연관 노트: [[빅프로젝트]])

---

## 1. 과제 개요 및 필요성
* **과제명**: 다목적 스마트시티 공간 의사결정시스템 (OmniSite)
* **목적**: 
  * B2G 행정 의사결정 시 부서 간 칸막이 병목을 제거하고, 공간정보(GIS) 분석과 LLM 다자 토론을 융합해 최적 입지 선정 및 타당성 결재 문서를 즉시 추출하는 행정 보조 도구 구축.
  * 기성 부스(개당 1,500만 원 상당) 설치 후 위법 시공 지적 및 주민 갈등으로 인한 강제 철거/이전(세금 낭비) 예산 매몰을 100% 방지.
* **실증 대상지 및 그라운드 트루스**: 
  * 실제 민원이 빈발하는 **서울특별시 용산구**를 실증 대상지로 포커싱.
  * 입지 예측의 신뢰도를 증명할 그라운드 트루스로 **'용산구 관내의 실제 불법 흡연 민원 데이터(E1)'**를 연동 및 팩트 검증.

## 🏥 2. 보건소 소관 모듈식 인프라 확장 구조 (Plug & Play)
단일 부서(지자체 보건소)에서 예산 일괄 편성 및 수의계약 발주가 가능하게 하여 B2G 사업성을 극대화했습니다.

1. **[MVP] 스마트 흡연부스 및 금연유도 CCTV 모듈**:
   * 버스/지하철 유동 거점 수요, 어린이집(30m)/학교(200m) 법적 절대 규제선 회피, E1 불법흡연 민원 가점 연동.
2. **[확장 1] 폭염/한파 실버 스마트 쉼터 모듈**:
   * 고령층 인구 밀집 주거지 수요, 혹서기/혹한기 기후 취약 어르신 동선 분석, 노인복지 조례 RAG 심의.
3. **[확장 2] 24시간 실외 스마트 AED(자동심장충격기) 응급구호 부스 모듈**:
   * 야간/공휴일 폐쇄 한계를 가진 건물 내부 AED의 사각지대 극복, **전국 2만 8,000대 KT 유휴 공중전화 부스 업사이클링 연계**, 응급의료법 RAG 심의.

## 🔄 3. 4단계 행정 의사결정 파이프라인 (User Flow)
지자체 폐쇄망(망분리) 환경을 완벽하게 만족시키기 위해 외부 실시간 API 호출을 배제하고 **오프라인 CSV 파일 임포트 방식**으로 파이프라인을 구축했습니다.

```
[공무원 데이터 업로드] ➔ [1단계: AI 데이터 감리] ➔ [2단계: PostGIS 공간 필터] ➔ [3단계: LangGraph RAG 심의] ➔ [4단계: PDF 결재 문서 인쇄]
```

1. **1단계 (감리)**: 지자체 내부망 수집 CSV를 브라우저에 업로드하면, **[1세트 AI 감리 에이전트]**가 스키마 무결성을 자가 검정합니다. (Vworld 지오코딩 API 매칭 신뢰도 90점 미만 시 경고 팝업을 띄워 사용자 보정(HITL)을 강제하고 불량 노이즈 원천 격리)
2. **2단계 (분석)**: 가중치 조작 시비(Gerrymandering)를 원천 차단하고자 일관성 비율(C.R. < 0.1) 검증을 통과한 가중치만 저장(**AHP Lock**)합니다. PostGIS 공간 인덱스(GIST)를 통해 100m 금역 차집합(`ST_Difference`) 및 소방로 3m, 보행 교행 1.2m 확보용 도로폭 5m 이상 필지를 **0.5초 이내**에 선별합니다.
3. **3단계 (조율)**: **[2세트 LangGraph RAG 에이전트]**(보건소 공무원, 상인대표, 시민대표)가 실제 자치구 조례를 RAG로 대조하며 팩트체크 기반 모의 찬반 토론을 거쳐 합의 조율 의견서를 작성합니다. (FastAPI SSE 단방향 웹소켓으로 말풍선 체감 지연 0.1초 렌더링)
4. **4단계 (결재)**: WeasyPrint 엔진을 기반으로, 지도, AHP 점수, AI 합의서와 함께 **가설건축물 축조 신고 자가진단표 및 도로점용료 자동 산식**이 결합된 공식 공문서 규격 PDF 보고서를 1일 이내로 즉시 다운로드 출력합니다.

## 📶 4. KT 그룹사 비즈니스 및 기술 시너지 (ARR & Lock-in)
* **KT 직접 고정 반복 매출 (ARR)**: 
  * 지자체 보건소의 분석 신뢰성 가점을 위한 **KT 생활인구 API 연간 라이선스 구독료 수입**.
  * 시공된 스마트 부스마다 원격 관제 및 IoT 안심벨 가동을 위해 보건소가 5~10년간 유지 납부하는 **KT B2G LTE-M/NB-IoT 통신 회선료**.
* **KT 그룹사 기술 인프라 락인**: 
  * 공공 보안성 심의 통과를 위해 플랫폼의 API 및 공간 DB 서버 배포지를 **KT Cloud 공공 Zone**으로 표준 지정.
  * 정보 외부 유출을 차단하기 위해 **KT의 한국어 특화 초거대 AI '믿음(Mi:dm)' sLLM**을 온프레미스 형태로 로컬 가동.
* **KT링커스 옥외광고(DOOH) 전환**: 
  * 골칫거리인 전국 유휴 공중전화 부스를 AED 부스로 개조 시 **KT Digico 디지털 사이니지**를 결합하여 B2B/B2G 광고 수익을 쉐어, 18개월 내 초기 시공비 전액 회수 구조 마련.

## 🚧 5. R&D 한계점 및 향후 고도화 로드맵
* **[공간 정보] 2D 평면 공간 분석의 한계**: 향후 국토부 **3D 브이월드(Vworld) 건물 모델 및 디지털 트윈**과 연계해 기류 시뮬레이션(CFD) 기반 실외 공기질/연기 수직 확산 및 빌딩풍 물리 분석 탑재 예정.
* **[인공 지능] 외부 LLM 의존 지연**: 공공 행정 문서 및 조례 특화 sLLM 자체 파인튜닝을 통해 외부 API 단가를 0원으로 낮추고 심의 레이턴시를 **5초 이내**로 단축.
* **[데이터셋] 정적 CSV 의존 한계**: **국민신문고 API 및 KT IoT 비상벨 호출 신호**와 Websocket/Kafka 스트림 실시간 연동을 통한 도시 라이브 모니터링 시스템 구축.
