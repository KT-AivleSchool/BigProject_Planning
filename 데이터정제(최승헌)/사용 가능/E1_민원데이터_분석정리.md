# E-1 민원 빅데이터 분석 정리

## 0. 목적과 위치
민원 데이터는 SDSS에서 **지점 핫스팟이 아니라 "흡연 갈등이 실재하는 도시문제"라는 배경 근거**로 사용한다.
(용산의 실제 갈등 지점은 A-1 담배꽁초 무단투기 8곳(지오코딩 완료)이 담당.)

- 사용 API: 2024 `minPttnStstAddrInfo` (키워드 기반 민원발생지별 정보)
- 방침: 넓은/좁은 검색어 각 1회 호출, target=`pttn,dfpt,saeol`(전체), 기간 2025년 전체

## 2. 해석 (배경 근거)
1. **검색어 노이즈 확인**: `금연구역`을 빼자 **은평구(4,150건)가 통째로 사라지고** 전체 건수가 급감.
   → `금연구역` 키워드가 주정차·현수막·불법광고 등 무관 민원까지 대량으로 끌어온 노이즈였음.
   → 흡연 갈등 지표로는 **좁은 검색어(간접흡연+담배꽁초)가 타당**.
2. **도심권 상위 패턴**: 좁은 검색어에서 서울은 **중구·강남·서초**가 상위.
   → 이들은 오피스·상업·역세권 밀집 자치구. **용산도 동일한 도심·역세권 특성**이라,
     이 상위군을 용산의 흡연 갈등 수요 벤치마크로 활용 가능.
3. **흡연 갈등은 광역적 도시문제**: 전국 상위가 대도시·도심 자치구에 집중 → 흡연구역 정비 수요의 보편성 입증.

---

## 3. 한계 (중요)
- **이 API는 전국 상위 9개만 반환**(2회 호출 모두 정확히 9건). topN 파라미터 없음.
- 따라서 **용산구는 전국 순위 밖 → 지점/자치구 값 직접 추출 불가.**
- 검색어를 좁혀도 전국 순위 경쟁이라 용산이 9위 내 진입 가능성 낮음.
- 결론: **본 API로 용산 특정값은 확보 불가.** 민원은 SDSS에서 배경지표로만 사용(당초 설계와 일치).

---

## 4. 향후 용산 특정값이 필요할 경우 (B안 경로)
- **국민신문고 헬프데스크(1600-8172)** 에 문의하여 **용산구청 mainSubCode**(이 API 전용 별도 기관코드, 7자리 추정) 확보.
- 스크립트(`fetch_minwon_yongsan.py`)의 `MAIN_SUB_CODE`에 입력 후 호출 → 용산구청 접수분 기준 건수 확보.
- 단 '처리기관 기준'이라 '발생지 기준'과 차이 있을 수 있음(근사값).

---

## 5. 재현 정보
- 스크립트: `fetch_minwon_yongsan.py`
- 파라미터: target=`pttn,dfpt,saeol`, dateFrom=`20250101`, dateTo=`20251231`, searchOption=`B0060005`, omitDuplicate=`false`
- 트래픽: 개발계정 100회/일 (본 분석 총 2회 사용)
- 라이선스: 제2유형(출처표시·상업적 이용금지) → 내부 배경분석용으로만 사용
